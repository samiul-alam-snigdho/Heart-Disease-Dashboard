# Heart Disease Analytics Dashboard

## 📊 Project Overview
An interactive data visualization dashboard designed to analyze patient survival rates, age demographics, and clinical risk factors (such as serum sodium levels and diabetes) associated with heart disease.

### Live Preview
![Dashboard Screenshot](image_12c7ea.jpg)

## 🛠️ Tools & Tech Stack
* **BI Platform: Power BI 
* **Data Transformation:Power Query
* **UX/UI Design: Custom Dark Mode Layout

## 📈 Key Metrics & Insights
* **Overall Survival: The cohort exhibits a 68% "Alive" rate, tracking 62 living patients against 132 total deaths.
* **Demographic Peaks: The 51–60 age group registers the highest volume of cases, followed closely by the 61–70 bracket.
* **Clinical Factors: Features integrated tracking for co-morbidities like diabetes mapped across specific age distributions.

## 🚀 How to Interact with the Project
1. Download the raw dashboard file (`Heart Disease Dashboard.pbix`) from this repository.
2. Open it using Power BI Desktop to utilize the interactive slicing features (e.g., filtering by Gender or Age).
